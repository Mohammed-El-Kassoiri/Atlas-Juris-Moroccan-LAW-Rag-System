# Frontend - Legal RAG Interface

React frontend for the Legal RAG system.

## Installation

```bash
npm install
```

## Configuration

Create a `.env` file in the frontend directory:

```
REACT_APP_API_URL=http://localhost:8000
```

## Running

```bash
npm start
```

The app will open at `http://localhost:3000`

## Features

- ✅ Modern, responsive UI
- ✅ Arabic/French language support
- ✅ Query interface
- ✅ Settings panel
- ✅ History with localStorage
- ✅ Show/hide prompt
- ✅ Resources display
- ✅ Real-time metrics

## Dependencies

- React 18
- Axios for API calls
- Lucide React for icons

## Build for Production

```bash
npm run build
```
